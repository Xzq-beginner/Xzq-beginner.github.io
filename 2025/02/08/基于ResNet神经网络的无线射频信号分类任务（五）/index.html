基于ResNet神经网络的无线射频信号分类任务（五）

基于ResNet神经网络的无线射频信号分类任务（五）

​ 在本节中，我将列举几种哪个可视化的分析方法，对分类效果进行检验。

一、绘制损失曲线

1. 绘制损失曲线

解读 ：

  • 使用 matplotlib 绘制训练损失曲线，横轴表示训练轮数，纵轴表示损失值。
  • plt.plot(range(epochs), losses) 绘制损失曲线，losses 是每轮训练的平均损失值。
  • label='训练损失' 为曲线设置标签。
  • markers='o' 在每个数据点处添加圆形标记。
  • plt.ylabel 和 plt.xlabel 分别设置纵轴和横轴的标签。
  • plt.title 设置图表的标题。
  • plt.legend 显示图例。
  • plt.grid 添加网格。
  • plt.text 在图表中添加文本信息，显示平均训练损失值。
  • plt.savefig 保存图表为图像文件。
  • plt.close() 关闭当前的图形，释放内存。

2. 功能

  • 直观展示模型在训练过程中的损失变化趋势。
  • 通过损失曲线可以观察模型是否收敛，以及是否存在过拟合或欠拟合等问题。

二、按信噪比绘制混淆矩阵

1. 获取所有唯一的信噪比

解读 ：

  • np.unique(all_snr_labels) 返回测试集中的所有唯一信噪比值。all_snr_labels 是在评估阶段收集的信噪比标签。

2. 按信噪比处理数据并绘制混淆矩阵

解读 ：

  • 使用 np.unique 获取所有唯一的信噪比值。
  • 使用 csv.DictWriter 创建一个 CSV 文件，用于存储不同信噪比下的性能指标。
  • 对于每个信噪比，筛选出对应的测试样本的标签和预测结果。
  • 使用 confusion_matrix 计算混淆矩阵，并使用 ConfusionMatrixDisplay 绘制混淆矩阵和归一化的混淆矩阵。
  • 计算并记录整体准确率、每类别的精确率、召回率和 F1 分数，以及宏平均和微平均指标。
  • 将计算得到的指标写入 CSV 文件，并通过日志记录打印结果。

3. 功能

  • 按信噪比分组，绘制不同信噪比下的混淆矩阵和归一化混淆矩阵，直观展示模型在不同信噪比条件下的分类性能。
  • 通过性能指标的计算和记录，可以定量分析模型在不同信噪比下的性能表现，以及各个类别之间的分类情况。

4.结果展示

三、绘制整体混淆矩阵

1. 计算整体混淆矩阵

解读 ：

  • 使用 confusion_matrix 计算测试集的整体混淆矩阵，all_labels 和 all_preds 分别是测试集的所有真实标签和预测标签。
  • 使用 ConfusionMatrixDisplay 创建混淆矩阵的显示对象。
  • 对混淆矩阵进行归一化处理，以便于比较各个类别之间的分类性能。

2. 绘制整体混淆矩阵

解读 ：

  • 使用 disp.plot 绘制混淆矩阵和归一化混淆矩阵。
  • cmap='Blues' 设置颜色映射，values_format 设置数值的显示格式。
  • xticks_rotation=45 旋转 x 轴的刻度标签，使其更易读。
  • 使用 plt.title 设置图表的标题，并用 plt.savefig 保存图表。

3. 功能

  • 创建并展示整体测试集的混淆矩阵和归一化混淆矩阵，直观展示模型的整体分类性能以及各个类别之间的分类情况。

4.结果展示

四、绘制热力图

1. 绘制整体混淆矩阵热力图

解读 ：

  • 使用 sns.heatmap 绘制热力图，annot=True 显示矩阵中的数值。
  • fmt='d' 设置数值的显示格式为整数。
  • xticklabels 和 yticklabels 设置 x 轴和 y 轴的刻度标签。
  • 使用 plt.xlabel 和 plt.ylabel 设置 x 轴和 y 轴的标签。

2. 绘制整体归一化混淆矩阵热力图

解读 ：

  • 绘制归一化混淆矩阵的热力图，fmt='.2f' 设置数值的显示格式为两位小数。
  • 其他部分与绘制整体混淆矩阵热力图类似。

3. 功能

  • 通过热力图的形式再次展示混淆矩阵和归一化混淆矩阵，帮助更直观地理解模型的分类性能。
  • 热力图中的颜色深浅表示数值的大小，方便观察矩阵中数值的分布情况。

五、保存模型

解读 ：

  • 使用 torch.save 保存整个模型到指定路径。
  • 模型文件名中包含了当前时间戳和测试集的准确率，方便区分不同时间点保存的模型。

功能

  • 将训练好的 ResNet34 模型保存到本地，以便后续使用或部署。

六、分析性能指标

1. 读取 CSV 文件并打印预览

解读 ：

  • 使用 pandas 的 read_csv 函数读取之前保存的 CSV 文件。
  • 使用 df.head() 查看数据的前几行，并通过日志记录打印出来。

2. 绘制趋势图

解读 ：

  • 使用 seaborn 的 lineplot 绘制整体准确率和每类别的 F1 分数随信噪比的变化趋势。
  • x='SNR' 和 y='Overall_Accuracy' 分别表示横轴和纵轴的变量。
  • marker='o' 在每个数据点处添加圆形标记。
  • plt.title 设置图表的标题，plt.xlabel 和 plt.ylabel 设置 x 轴和 y 轴的标签。
  • 使用 plt.grid 添加网格，plt.savefig 保存图表。

3. 找出整体性能最佳的信噪比

解读 ：

  • 使用 df.loc[df['Overall_Accuracy'].idxmax()] 找到整体准确率最高的行，进而获取对应的信噪比和准确率。
  • 通过日志记录打印出最佳信噪比及其对应的准确率。

4. 分析每类别的平均性能

解读 ：

  • 计算每类别的 F1 分数的平均值，使用 df.mean() 实现。
  • 通过日志记录打印出每类别的平均 F1 分数。

功能

  • 分析并展示模型在不同信噪比下的整体准确率和每类别的 F1 分数的变化趋势。
  • 找出模型在哪个信噪比下性能最佳，并分析每类别的平均性能，为进一步优化模型提供依据。
