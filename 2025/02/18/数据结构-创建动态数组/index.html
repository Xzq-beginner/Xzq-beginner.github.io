数据结构-创建动态数组

基于Java的动态数组的实现

​ 在 Java 开发中，ArrayList 是我们常用的动态数组实现，它提供了灵活的增删改查操作。然而，深入了解其底层实现不仅能帮助我们更好地使用它，还能提升我们对数据结构的理解。因此，我决定自己实现一个简单的动态数组——DynamicArrays，并在此分享我的实现细节和设计思路。

1. 动态数组的整体设计

​ DynamicArrays 是一个基于数组实现的动态数据结构，它支持以下功能：

  • 增删改查：在指定位置插入、删除、修改和查询元素。
  • 自动扩容：当数组容量不足时，自动调整容量。
  • 遍历与流操作：支持通过迭代器和流的方式遍历数组。
  • 排序与统计：提供排序功能，并支持统计元素出现次数。

下面，我将详细介绍每个方法的实现和用途。

2. 核心字段与构造

​ 在 DynamicArrays 中，我们定义了以下核心字段：

  • capacity：表示数组的当前容量，初始值为 10。
  • array：实际存储数据的数组，初始为空数组。
  • size：表示数组中当前存储的有效元素个数。

3. 动态数组的核心方法

3.1. addLast(int element)：向数组末尾添加元素

功能描述：
该方法用于向动态数组的末尾添加一个元素。如果数组容量不足，会自动调用 expansion() 方法扩容。

设计思路：

  • 在添加元素前，先检查数组是否已满（size == capacity）。如果已满，则调用 expansion() 方法扩容。
  • 将新元素添加到数组的最后一个位置（array[size]），并更新 size。

3.2. add(int index, int element)：在指定索引位置插入元素

功能描述：
该方法用于在指定索引位置插入一个元素。如果索引越界，抛出 IndexOutOfBoundsException。

设计思路：

  • 首先检查索引是否有效（0 <= index <= size）。
  • 使用 System.arraycopy() 将插入位置后的元素向后移动一位，为新元素腾出空间。
  • 将新元素插入到指定位置，并更新 size。

3.3. remove(int index)：删除指定索引位置的元素

功能描述：
该方法用于删除指定索引位置的元素。如果索引越界，抛出 IndexOutOfBoundsException。

设计思路：

  • 首先检查索引是否有效（0 <= index < size）。
  • 使用 System.arraycopy() 将删除位置后的元素向前移动一位，覆盖掉被删除的元素。
  • 更新 size。

3.4. removeInquire(int element)：删除指定值的第一个匹配元素

功能描述：
该方法用于删除数组中第一个匹配指定值的元素。如果有多个相同元素，只会删除第一个出现的。

设计思路：

  • 遍历数组，找到第一个匹配的元素。
  • 调用 remove(int index) 方法删除该元素。

3.5. clear()：清空数组

功能描述：
该方法用于清空数组中的所有元素，并重置容量为初始值（10）。

3.6. set(int index, int element)：修改指定索引位置的元素

功能描述：
该方法用于修改指定索引位置的元素。如果索引越界，抛出 IndexOutOfBoundsException。

3.7. get(int index)：查询指定索引位置的元素

功能描述：
该方法用于查询指定索引位置的元素。如果索引越界，抛出 IndexOutOfBoundsException。

3.8. inquire(int element)：查询指定元素的索引

功能描述：
该方法用于查询数组中是否存在指定的元素。如果存在，返回第一个匹配元素的索引；否则返回 -1。

3.9. traverse()：遍历数组并打印元素

功能描述：
该方法用于遍历数组中的所有元素，并逐行打印每个元素。

3.10. foreach(Consumer<Integer> consumer)：使用函数式接口遍历数组

功能描述：
该方法使用传入的 Consumer 函数式接口对数组中的每个元素执行操作。

3.11. iterator()：返回迭代器

功能描述：
该方法返回一个迭代器，用于遍历数组中的元素。

3.12. stream()：返回 IntStream

功能描述：
该方法返回一个 IntStream，包含数组中的有效元素。

3.13. count(int element)：统计元素出现次数

功能描述：
该方法用于统计数组中指定元素出现的次数。

3.14. sort()：对数组排序

功能描述：
该方法对动态数组中的元素进行排序。根据数组大小自动选择排序算法：

  • 如果数组元素个数小于 100，使用冒泡排序。
  • 否则，使用快速排序。

3.15. expansion()：自动扩容

java复制

功能描述：
该方法用于检查数组容量是否足够，并在必要时扩容。当数组的有效元素个数等于数组容量时，数组自动扩容为原来的 1.5 倍。

4. 总结

​ 这个实现不仅支持常见的增删改查操作，还提供了自动扩容、排序、遍历等功能。希望这篇博客能帮助你更好地理解动态数组的实现原理，并在实际开发中灵活应用。
